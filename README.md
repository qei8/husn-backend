# نظام حصن (Hisn) 🚁🔥

نظام متكامل يعتمد على تقنيات الذكاء الاصطناعي والطائرات بدون طيار (UAV) للكشف المبكر عن حرائق الغابات، مما يساهم في سرعة الاستجابة وتقليل الأضرار البيئية.

## 📁 هيكلية المشروع (Project Structure)

يحتوي هذا المستودع على المكونات الرئيسية التالية للنظام:

* **`husn-backend/`**: يحتوي على ملفات الخادم (Back-end) لمعالجة البيانات والربط مع قواعد البيانات والذكاء الاصطناعي.
* **`husn-frontend/`**: يحتوي على واجهة المستخدم (Front-end) لعرض البيانات والتنبيهات وحالة الطائرة.
* **`drone-simulator.js`**: سكريبت مخصص لمحاكاة حركة طائرة الدرون والبيانات الصادرة منها (مثل الإحداثيات وحالة الكاميرا) لأغراض الاختبار والتطوير.

## ⚙️ المتطلبات الأساسية (Prerequisites)

قبل البدء في تشغيل المشروع، تأكد من تثبيت البرامج التالية على جهازك:
* [Node.js و npm](https://nodejs.org/)
